# PG#42626 - 더 맵게

## 느낀점
* heapify 함수는 리턴값이 없다.
* 엣지케이스를 따져보기 좋은 문제.

## input
```
scoville : [1, 2, 3, 9, 10, 12] // 
K : 7
```

## output
```
2 // 모든 음식의 스코빌 지수를 7 이상으로 만들기 위해선 최소 두 번 섞어야함.
```

## notes
* 섞 = 가장안맵 + (그다음안맵 * 2)
* 불가한 경우 -1 리턴

## strategy
* 가장 덜매운 걸 섞는다고 했는데
* 가장 작은 두개를 꺼내서 섞은담에 넣고... 반복하다가
* 딱 꺼냈는데 K 이상이면 그 즉시 리턴
