# PG#42883 - 큰 수 만들기

## 느낀점
* 

## input
```
number
'1924'

k
2

```
1924에서 k개의 수를 제거했을 때 얻을 수 있는 가장 큰 수는

## output
```
'94'
```
94임.

## notes
* 

## strategy
* 필승법 bf는 뭐 당연히 있고
* 더 효율적으로 할 수 있을텐데
  * n개 중 큰 수 k개만 남기고, n-k개는 제끼면 됨.
  * 그러면 enumerate 후 sort를 해서
  * 큰 값들만 새로운 arr에 배치한 다음에
  * join하면 되겠다.
