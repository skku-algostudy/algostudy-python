# PG#42839 - 소수 찾기

## 느낀점
* 팀노트에 is_prime 함수 추가해도 좋을듯. -> 근데 쉽게 구현 가능해서, 안해도 될것같다.
* 예전에 못풀었던건 아마 itertools의 존재를 몰랐기 때문인듯.

## input
```
numbers
"17"
```
1과 7이라는 종이조각이 있을 떄

## output
```
3
```
7, 17, 71의 소수를 만들 수 있음.
