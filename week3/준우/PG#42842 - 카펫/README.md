# PG#42842 - 카펫

## 느낀점
* 

## input
```
brown
10

yellow
2


```
갈색칸 10개, 노란칸 2개가 있다면

## output
```
return
[4,3]


```
가로 4 세로 3임.

## notes
* 가로 >= 세로
* 테두리는 갈색, 중앙은 노란색

## strategy
* 음.. 24 24의 경우의 수를 보자.
* 노란색의 경우의 수는
  * 24 1 -> 갈색은 26 * 2 + 1 * 2
  * 12 2 -> 갈색은 14 * 2 + 2 * 2
  * 8 3 -> ...
  * 6 4 -> 갈색은 8 * 2 + 4 * 2 : 24 요게 정답
